パララックススクロール効果

作者：Vineet Kumar

パララックススクロールとは？

パララックススクロールは、背景を前景よりもゆっくり動かすことで、スクロール時に奥行きを与え、より没入感のある閲覧体験を生み出すWebデザイン手法です。視差は光学的な錯視に基づいており、目は近くの物体を大きく、遠くの物体を小さく認識するため、遠くのものがゆっくり動いているように感じます。この錯視はさまざまなメディアのパララックス効果で長く活用され、よりリアルな表現を作り出してきました。最初の例はディズニーの『白雪姫』のような伝統的なアニメーションや、『スーパーマリオ』などのビデオゲームにまでさかのぼります。CSS と HTML の進化に伴い、パララックス効果は今日親しまれているWebデザイン手法へと発展しました。

2007年、開発者の Glutnix がブログで初期のブラウザ向けパララックス実装の一つを共有し、JavaScript と CSS2 のサンプルコードとデモを公開しました。当時の IE6 などにも対応していましたが、2011年に HTML5 と CSS3 が登場してから、この技術は本格的にWebデザイナーに支持されるようになりました。支持者は、パララックス背景を使えばサイトの流動性を活かしつつユーザーを惹きつけ、全体の体験を高められると考えています。しかし、2013年にパデュー大学が発表した研究は「…パララックススクロールは一部のユーザー体験を向上させるが、必ずしも全体の体験を高めるとは限らない」と指摘しています。


なぜパララックス画像を使うのか？

    デザイナーがパララックス画像を試すべき主な理由は2つあります。

  • ストーリーテリングに役立つ

    パララックス画像は物語を語るのに最適です。優れたデザインの本質はコミュニケーションにあり、サイト制作では訪問者と対話し、製品・サービス・ブランドのストーリーを伝えたいと考えます。パララックススクロールは訪問者を独特のオンライン体験に没入させ、単一ページでもブランドのストーリーを完結に描けます。

  • ユーザーのエンゲージメントを高める

    パララックス画像はユーザーとのやり取りを促進します。サイト設計では直帰率を抑えることが重要です。直帰率は1ページだけ見て「戻る」かタブを閉じた訪問者の割合で、高いと検索順位にも影響します。丁寧に作り込まれたパララックスページなら、動きのある体験でユーザーがより長く滞在してくれます。


サイトにパララックス効果を加えるときのヒント：

  • ポイントを絞って使う

    パララックススクロールはすべてのページや状況で使うより、特定の要素に絞ると効果的です。ヒーローや見出し部分、あるいはトップページだけに配置して、デザインを際立たせることを目的にしましょう。

  • 画像を圧縮する

    パララックススクロールではさまざまなメディアやCSSシェイプを使って奥行きを演出します。 読み込みを重くしないよう、大きすぎる画像は避けて最適化しましょう。

  • 色彩を上手に使う

    パララックス効果は前景と背景の画像だけでなく、配色でも深みを出せます。背景は彩度を落とし、前景は鮮やかな色を使うと、レイヤー感を一層際立たせられます。
